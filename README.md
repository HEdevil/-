碟片租赁商店管理系统
===
简介
-----
>本项目是为了熟练掌握CPP面向对象编程的思想而做的练习，该项目为碟片租赁商店进行了设计，店主使用这套系统可以在客户在DVD商店进行消费后，根据用户所借的不同的DVD种类和数量来计算用户消费的金额和积分。
